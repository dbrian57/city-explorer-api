# City Explorer API

**Author**: Dan Brian
**Version**: 1.0.0

## Overview
This is an API that provided weather updates for selected cities.

## Getting Started
If you want to hit the API, type this end point into your browser: `/weather?searchQuery=Seattle` to return the forcast for Seattle.
